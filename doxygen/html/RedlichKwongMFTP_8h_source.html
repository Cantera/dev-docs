90  /**
91  * Calculate the density of the mixture using the partial molar volumes and
92  * mole fractions as input
93  *
94  * The formula for this is
95  *
96  * \f[
97  * \rho = \frac{\sum_k{X_k W_k}}{\sum_k{X_k V_k}}
98  * \f]
99  *
100  * where \f$X_k\f$ are the mole fractions, \f$W_k\f$ are the molecular
101  * weights, and \f$V_k\f$ are the pure species molar volumes.
102  *
103  * Note, the basis behind this formula is that in an ideal solution the
104  * partial molar volumes are equal to the species standard state molar
105  * volumes. The species molar volumes may be functions of temperature and
106  * pressure.
107  */

269  //! Solve the cubic equation of state
270  /*!
271  * The R-K equation of state may be solved via the following formula:
272  *
273  * V**3 - V**2(RT/P) - V(RTb/P - a/(P T**.5) + b*b) - (a b / (P T**.5)) = 0
274  *
275  * Returns the number of solutions found. If it only finds the liquid
276  * branch solution, it will return a -1 or a -2 instead of 1 or 2. If it
277  * returns 0, then there is an error.
278  */
279  int NicholsSolve(double TKelvin, double pres, doublereal a, doublereal b,
