21 //! Class StoichSubstance represents a stoichiometric (fixed composition)
22 //! incompressible substance.
23 /*!
24  * This class internally changes the independent degree of freedom from density
25  * to pressure. This is necessary because the phase is incompressible. It uses a
26  * constant volume approximation.
27  *
28  * <b> Specification of Species Standard State Properties </b>
29  *
30  * This class inherits from SingleSpeciesTP. It is assumed that the reference
31  * state thermodynamics may be obtained by a pointer to a populated species
32  * thermodynamic property manager class (see ThermoPhase::m_spthermo). How to
33  * relate pressure changes to the reference state thermodynamics is resolved at
34  * this level.
35  *
36  * For an incompressible, stoichiometric substance, the molar internal energy is
37  * independent of pressure. Since the thermodynamic properties are specified by
38  * giving the standard-state enthalpy, the term \f$ P_0 \hat v\f$ is subtracted
39  * from the specified molar enthalpy to compute the molar internal energy. The
40  * entropy is assumed to be independent of the pressure.
41  *
42  * The enthalpy function is given by the following relation.
43  *
44  * \f[
45  * h^o_k(T,P) =
46  * h^{ref}_k(T) + \tilde v \left( P - P_{ref} \right)
47  * \f]
48  *
49  * For an incompressible, stoichiometric substance, the molar internal energy is
50  * independent of pressure. Since the thermodynamic properties are specified by
51  * giving the standard-state enthalpy, the term \f$ P_{ref} \tilde v\f$ is
52  * subtracted from the specified reference molar enthalpy to compute the molar
53  * internal energy.
54  *
55  * \f[
56  * u^o_k(T,P) = h^{ref}_k(T) - P_{ref} \tilde v
57  * \f]
58  *
59  * The standard state heat capacity and entropy are independent of pressure. The
60  * standard state Gibbs free energy is obtained from the enthalpy and entropy
61  * functions.
62  *
63  * <b> Specification of Solution Thermodynamic Properties </b>
64  *
65  * All solution properties are obtained from the standard state species
66  * functions, since there is only one species in the phase.
67  *
68  * <b> Application within Kinetics Managers </b>
69  *
70  * The standard concentration is equal to 1.0. This means that the kinetics
71  * operator works on an (activities basis). Since this is a stoichiometric
72  * substance, this means that the concentration of this phase drops out of
73  * kinetics expressions.
74  *
75  * An example of a reaction using this is a sticking coefficient reaction of a
76  * substance in an ideal gas phase on a surface with a bulk phase species in
77  * this phase. In this case, the rate of progress for this reaction,
78  * \f$ R_s \f$, may be expressed via the following equation:
79  * \f[
80  * R_s = k_s C_{gas}
81  * \f]
82  * where the units for \f$ R_s \f$ are kmol m-2 s-1. \f$ C_{gas} \f$ has units
83  * of kmol m-3. Therefore, the kinetic rate constant, \f$ k_s \f$, has units of
84  * m s-1. Nowhere does the concentration of the bulk phase appear in the rate
85  * constant expression, since it's a stoichiometric phase and the activity is
86  * always equal to 1.0.
87  *
88  * <b> Instantiation of the Class </b>
89  *
90  * The constructor for this phase is NOT located in the default ThermoFactory
91  * for %Cantera. However, a new StoichSubstance may be created by
92  * the following code snippets:
93  *
94  * @code
95  * XML_Node *xm = get_XML_NameID("phase", iFile + "#NaCl(S)", 0);
96  * StoichSubstance *solid = new StoichSubstance(*xm);
97  * @endcode
98  *
99  * or by the following call to importPhase():
100  *
101  * @code
102  * XML_Node *xm = get_XML_NameID("phase", iFile + "#NaCl(S)", 0);
103  * StoichSubstance solid;
104  * importPhase(*xm, &solid);
105  * @endcode
106  *
107  * <b> XML Example </b>
108  *
109  * The phase model name for this is called StoichSubstance. It must be supplied
110  * as the model attribute of the thermo XML element entry. Within the phase XML
111  * block, the density of the phase must be specified. An example of an XML file
112  * this phase is given below.
113  *
114  * @code
115  * <!-- phase NaCl(S) -->
116  * <phase dim="3" id="NaCl(S)">
117  * <elementArray datasrc="elements.xml">
118  * Na Cl
119  * </elementArray>
120  * <speciesArray datasrc="#species_NaCl(S)"> NaCl(S) </speciesArray>
121  * <thermo model="StoichSubstance">
122  * <density units="g/cm3">2.165</density>
123  * </thermo>
124  * <transport model="None"/>
125  * <kinetics model="none"/>
126  * </phase>
127  *
128  * <!-- species definitions -->
129  * <speciesData id="species_NaCl(S)">
130  * <!-- species NaCl(S) -->
131  * <species name="NaCl(S)">
132  * <atomArray> Na:1 Cl:1 </atomArray>
133  * <thermo>
134  * <Shomate Pref="1 bar" Tmax="1075.0" Tmin="250.0">
135  * <floatArray size="7">
136  * 50.72389, 6.672267, -2.517167,
137  * 10.15934, -0.200675, -427.2115,
138  * 130.3973
139  * </floatArray>
140  * </Shomate>
141  * </thermo>
142  * <density units="g/cm3">2.165</density>
143  * </species>
144  * </speciesData> @endcode
145  *
146  * The model attribute, "StoichSubstance", on the thermo element
147  * identifies the phase as being a StoichSubstance object.
148  *
149  * @ingroup thermoprops
150  */
