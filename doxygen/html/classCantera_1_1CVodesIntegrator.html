Detailed Description

Wrapper class for 'cvodes' integrator from LLNL.

See Also
FuncEval.h. Classes that use CVodeInt: ImplicitChem, ImplicitSurfChem, Reactor
Deprecated:
Support for Sundials 2.2 and 2.3 is deprecated. Starting with Cantera 2.2, only Sundials versions 2.4 and newer will be supported. (The CVodesIntegrator class itself is not deprecated).
