20 /**
21  * The NASA polynomial parameterization for two temperature ranges. This
22  * parameterization expresses the heat capacity as a fourth-order polynomial.
23  * Note that this is the form used in the 1971 NASA equilibrium program and by
24  * the Chemkin software package, but differs from the form used in the more
25  * recent NASA equilibrium program.
26  *
27  * Seven coefficients \f$(a_0,\dots,a_6)\f$ are used to represent
28  * \f$ c_p^0(T)\f$, \f$ h^0(T)\f$, and \f$ s^0(T) \f$ as
29  * polynomials in \f$ T \f$ :
30  * \f[
31  * \frac{c_p(T)}{R} = a_0 + a_1 T + a_2 T^2 + a_3 T^3 + a_4 T^4
32  * \f]
33  * \f[
34  * \frac{h^0(T)}{RT} = a_0 + \frac{a_1}{2} T + \frac{a_2}{3} T^2
35  * + \frac{a_3}{4} T^3 + \frac{a_4}{5} T^4 + \frac{a_5}{T}.
36  * \f]
37  * \f[
38  * \frac{s^0(T)}{R} = a_0\ln T + a_1 T + \frac{a_2}{2} T^2
39  * + \frac{a_3}{3} T^3 + \frac{a_4}{4} T^4 + a_6.
40  * \f]
41  *
42  * This class is designed specifically for use by the class
43  * GeneralSpeciesThermo.
44  *
45  * @ingroup spthermo
46  */
